Fixes all known post content issues on WD Satellites.

== Description ==

The WD Bulk Content Fixer plugin scans all posts and pages and fixes all known content issues on WD Satellites.

== Installation ==

Note: This plugin requires the WordPress 5+ editor.

1. Upload 'wd-bulk-validation-fixer' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Tools > WD Content Fixer.
4. Click the 'Scan Content' button.
5. Optionally convert individual posts or bulk convert all.
